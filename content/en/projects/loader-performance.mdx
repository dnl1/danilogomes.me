## Problem description

The product had poor Lighthouse scores on mobile and high bounce rates on slow networks.

## Architecture

I moved dynamic blocks to static generation with selective hydration, split heavy modules, and added immutable
edge caching for media assets.

## Impact

- Largest Contentful Paint reduced from 3.8s to 1.7s
- Bounce rate dropped by 19%
- Build output became mostly static and cheaper to host
