Um portfolio orientado a conteudo deve otimizar duas coisas:

1. Velocidade de publicacao
2. Sustentabilidade no longo prazo

Com **App Router + MDX**, voce mantem o conteudo perto do codigo sem introduzir um CMS completo antes da hora.

## Por que essa estrutura funciona

- Geracao estatica mantem o custo de hospedagem baixo
- MDX permite conteudo rico e snippets de codigo
- Metadados por rota ajudam SEO e compartilhamento social

```tsx
export async function generateStaticParams() {
  const slugs = await getSlugs("pt", "blog");
  return slugs.map((slug) => ({ slug }));
}
```

Quando o volume de conteudo crescer, voce pode migrar para um headless CMS preservando a logica de rota e renderizacao.
