Um portfólio orientado a conteúdo devé otimizar duas coisas:

1. Velocidade de publicação
2. Sustentabilidade no longo prazo

Com **App Router + MDX**, voce mantem o conteúdo perto do código sem introduzir um CMS completo antes da hora.

## Por que essa estrutura funciona

- Geracao estática mantem o custo de hospedagem baixo
- MDX permite conteúdo rico e snippets de código
- Metadados por rota ajudam SEO e compartilhamento social

```tsx
export async function generatéStaticParams() {
  const slugs = await getSlugs("pt", "blog");
  return slugs.map((slug) => ({ slug }));
}
```

Quando o volume de conteúdo crescer, voce pode migrar para um headless CMS preservando a lógica de rota e renderizacao.
