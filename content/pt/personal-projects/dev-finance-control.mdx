## Overview

Dev Finance Control é um sistema full stack em Next.js criado para ajudar desenvolvedores e profissionais independentes
a organizar finanças pessoais, renda por projeto, invoices e fluxos de nota fiscal com mais clareza do que apps
financeiros de uso geral costumam oferecer.

## Motivation

Freelancers e contractors frequentemente recebem valores de vários clientes, plataformas diferentes e, em alguns casos,
moedas distintas. A maioria dos aplicativos financeiros foi pensada para orçamento doméstico, e não para pessoas cuja
renda depende de projetos, invoices e ciclos variáveis de entrega.

## Problem

A organização financeira se torna difícil quando a renda profissional vem de várias fontes. Os pontos mais problemáticos
incluíam:

- acompanhar pagamentos de vários clientes
- organizar registros financeiros por contexto de projeto
- controlar despesas relacionadas ao trabalho
- gerar invoices e notas fiscais dentro do fluxo de cobrança
- manter visibilidade clara da saúde financeira geral

## Solution

Dev Finance Control oferece uma plataforma financeira focada em desenvolvedores que precisam entender renda, despesa,
invoices e fluxos de nota fiscal sem carregar a complexidade desnecessária de apps financeiros convencionais, tudo
dentro de uma única aplicação full stack em Next.js.

Funcionalidades principais:

- controle de receitas
- controle de despesas
- dashboards financeiros
- organização de renda por projeto
- geração de invoices
- geração de nota fiscal

## Technical Challenges

O produto exigia mais do que telas de CRUD. Ele precisava de modelos flexíveis para representar tipos diferentes de
transação, documentos de cobrança e registros fiscais, sem perder clareza nos dashboards dentro de uma única aplicação
full stack.

Principais desafios técnicos:

- desenhar modelos financeiros flexíveis para diferentes categorias de transação
- organizar transações por contexto de projeto e renda
- suportar geração de invoice e nota fiscal dentro do mesmo fluxo financeiro
- estruturar backend e frontend de forma limpa dentro de uma aplicação full Next.js
- criar dashboards que aumentassem visibilidade sem sobrecarregar a interface
- manter a UX intuitiva preservando utilidade analítica

## Technologies Used

- Next.js
- TypeScript
- Vercel
